# Excel Practice Questions (Beginner → Advanced)

## 🔹 Level 1: Basics

1. Create a table with columns:
   - Name, Marks1, Marks2, Marks3
   - Calculate Total using SUM
   - Calculate Average using AVERAGE

2. Find:
   - Maximum marks
   - Minimum marks

3. Generate 5 random numbers using RAND()

---

## 🔹 Level 2: Logical Functions

4. Use IF:
   - If marks >= 40 → "Pass"
   - Else → "Fail"

5. Use nested IF:
   - >= 75 → Distinction
   - >= 60 → First Class
   - >= 40 → Pass
   - Else → Fail

6. Use AND:
   - If Marks1 > 40 AND Marks2 > 40 → "Eligible"

7. Use OR:
   - If any subject < 40 → "Reappear"

8. Use XOR:
   - Check if exactly one condition is true

---

## 🔹 Level 3: Counting Functions

9. Use COUNT (numbers only)

11. Use COUNTBLANK
12. Use COUNTIF:

13. Use COUNTIFS:
    - Count students with Marks1 > 50 AND Marks2 > 50

## 🔹 Level 4: Conditional Calculations
